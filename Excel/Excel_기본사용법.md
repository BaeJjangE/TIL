# Excel

## 환경 설정

```bash
# 기본 글꼴 설정하기
- 파일 -> 옵션 -> 일반 -> 새 통합 문서 만들기 -> 글꼴 설정

# 기본 저장 시간 주기 설정하기
- 파일 -> 옵션 -> 저장 -> 통합문서저장 -> 자동 복구 정보 저장 간격 설정

# 하이퍼링크 입력 설정하기
- 파일 -> 옵션 -> 언어 교정 -> 자동 고침 옵션 -> 입력할 때 자동 서식 -> 하이퍼링크 설정

# 특수기호 편리하게 입력하는 방법: 자동 고침 기호
- 파일 -> 옵션 -> 언어 교정 -> 자동 고침 옵션 -> 자동 고침 -> 입력 과 결과에 입력
ex) 실제 사용 시 입력 후 띄어쓰기하면 적용
ㅁ별 -> ★(ㅁ한자8)
ㅁㅁ -> ■(ㅁ한자)
ㅁ> -> ▶(ㅁ한자)
ㅁ감소 -> ▼(ㅁ한자)
- 오빠두엑셀 책 p. 27 참고

- 자동 고침 기호 일괄등록 서식
 - 밑에 링크에서 다운받아서 실행하면 매크로 자동 적용
```
